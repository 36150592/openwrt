 - MC9090_generic_filter_CDMA.sqf
 - Description - To be used as the generic filter when raising any of the

                 - Technology selection related issues (1xRTT to EVDO for CDMA)
                 - QMI call failures
                 - OMADM for CDMA
                 - RUIM
